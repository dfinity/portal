## Integrate with Plug wallet

[Plug wallet](https://www.plug-wallet.com/) uses Secp256k1 as its key algorithm. To integrate Plug wallet authentication into your dapp, you can use the `@dfinity/identity-secp256k1` and a PEM file:

```js
import { Secp256k1KeyIdentity } from '@dfinity/identity-secp256k1';

const identity = Secp256k1KeyIdentity.fromPem(pemFileContentString);
```
