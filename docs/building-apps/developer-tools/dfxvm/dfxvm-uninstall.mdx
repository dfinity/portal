# dfxvm uninstall

<MarkdownChipRow labels={["Reference"]} />

Uninstalls a version of dfx.

## Usage

```bash
dfxvm uninstall <version>
```

## Examples

Uninstall a beta that you don't want anymore:

```bash
dfxvm uninstall 0.15.1-beta.0
```
