# Tabs

The `AdornedTabs` component provides you with a mechanism to insert a start or
end adornment to any of its `AdornedTab` children.

`startAdornment`: inserts any element before the tab label `endAdornment`:
inserts any element after the tab label

Both `startAdornment` and `endAdornment` may be used together.

<AdornedTabs groupId="tab-group">
  <AdornedTab value="1" label="First tab" default startAdornment={(
    <Chip size={"small"} shape={"rounded"} color={"bg-black"}>
      First
    </Chip>
  )}>
