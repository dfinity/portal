## Overview

This section will give you an overview of what data can be fetched from the data API endpoints for the ICP Rosetta implementation. 

It is split into the following subsections:

- [Network](/docs/developer-docs/defi/rosetta/icp_rosetta/data_api/network.mdx): Fetch network specific information, such as the most recent block and what network rosetta is connected to. 
- [Balances](/docs/developer-docs/defi/rosetta/icp_rosetta/data_api/balances.mdx): Fetch the balance of an account_identifier or neuron.
- [Blocks](/docs/developer-docs/defi/rosetta/icp_rosetta/data_api/blocks.mdx): Fetch blocks from the ICP ledger. 
- [Transactions](/docs/developer-docs/defi/rosetta/icp_rosetta/data_api/transactions.mdx): Fetch transactions from the ICP ledger.
- [Known neurons](/docs/developer-docs/defi/rosetta/icp_rosetta/data_api/list_known_neurons.mdx): Fetch a list of all publicly known neurons. 
- [Pending proposals](/docs/developer-docs/defi/rosetta/icp_rosetta/data_api/get_pending_proposals.mdx): Fetch a list of all currently pending proposals on the NNS.
- [Proposal info](/docs/developer-docs/defi/rosetta/icp_rosetta/data_api/get_proposal_info.mdx): Fetch the information about a specific proposal on the NNS.
