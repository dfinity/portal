# SNS ledger canister
## Overview

The SNS ledger canister keeps track of the transactions between accounts in a **chain**,
as well as the account balances at the last block of the chain.
A ledger canister is always deployed as part of the SNS canisters. 
The ledger canister follows the [ICRC-1](https://github.com/dfinity/ICRC-1) specification.

For instructions on how to install the ICRC-1 Ledger follow [ICRC-1 Ledger Deployment](/docs/current/developer-docs/defi/icrc-1/icrc1-ledger-setup).
For instructions on how to interact with the ICRC-1 Ledger follow [ICRC-1 Ledger Interaction](/docs/current/developer-docs/defi/icrc-1/using-icrc1-ledger).
