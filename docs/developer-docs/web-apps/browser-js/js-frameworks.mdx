## Overview

To build user interfaces that utilize in-browser JavaScript, several frameworks can be used.

## React

[React](/docs/current/developer-docs/web-apps/application-frontends/quickstart/react-quickstart): React is a library for creating user interfaces, and is supported as a default frontend development option within the `dfx new` command (`dfx` versions v0.17.0 and newer). You can learn more about React in the [React quickstart guide](/docs/current/developer-docs/web-apps/application-frontends/quickstart/react-quickstart).

Other React examples include:

- [Vite + React + Motoko](https://github.com/rvanasa/vite-react-motoko).

- [Create-ic-app: React template](https://github.com/MioQuispe/create-ic-app/tree/main/template-react).

- [React to-do sample](https://github.com/ORIGYN-SA/todos-react-native-dfinity).

- [React + ReactJS + React-Router](https://github.com/rbolog/dfinity_reactJs_reactRouter_babel).

- [Vite + React + Material UI](https://github.com/ferMartz/ic-firestarter).

## Next.js

[Next.js](/docs/current/developer-docs/web-apps/application-frontends/existing-frontend/#nextjs): Next.js is an open-source framework that provides React-based apps with server-side website rendering and generation. Next.js can be used in an ICP application by generating static Next.js application files and pointing your frontend canister to those static assets. [Learn more in the Next.js documentation](/docs/current/developer-docs/web-apps/application-frontends/existing-frontend/#nextjs).

Other Next.js examples include:

- [Next.js + BlankUI](https://github.com/Magvin/nextjs-icp).

- [Next.js dapp example](https://github.com/warashibe/nextdapp).

- [Next.js IC starter template](https://github.com/beyond009/nextjs-ic-starter).

- [Next.js calculator dapp template](https://github.com/gbonumore/index-calculator).


## Vue

[Vue](https://vuejs.org/) is a progressive JavaScript framework commonly used for building single-page applications. Some examples using Vue include:

- [Vue integration with `dfx`'s webpack bootstrap](https://github.com/nop33/dfinity-vue).

- [Create-ic-app: Vue template](https://github.com/MioQuispe/create-ic-app/tree/main/template-vue).


## Svelte

[Svelte](https://svelte.dev/) is a compact JS framework for building detailed frontend components. It is supported as a default frontend development option within the `dfx new` command (`dfx` versions v0.17.0 and newer). Another example using Svelte is:

- [Create-ic-app: Svelte template](https://github.com/MioQuispe/create-ic-app/tree/main/template-svelte).
